Java: Operadores

Aprende a utilizar todos los operadores lógicos.

Operadores en Java

En Java existen diferentes tipos de operadores, simbolo o secuencias que podemos utilizar para realizar operaciones con diferentes tipos de datos y/o variables.

Operadores Aritméticos

Los operadores aritméticos son aquellos que utilizamos para realizar operaciones matemáticas, con variables o datos numéricos.

public class Main {
    public static void main(String[] args) {
        int a = 5;
        int b = 3;
        System.out.println(a + b);//Suma
        System.out.println(a - b);//Resta
        System.out.println(a * b);//Multipliucación
        System.out.println(a / b);//División
        System.out.println(a % b);//Resto de una División
    }
}

Operadores de Asignación

Son los operadores que utilizamos para asignar valores a variables. Que también podemos utilizar para cambiar el valor de una variable de cierta forma. Realizar una operación con su mismo valor y guardarlo de nuevo en la variable.

public class Main {
    public static void main(String[] args) {
        int a = 5;
        int b = 3;
        
        a = b;
        System.out.println(a);
        
        a += 1; // a = a + 1
        System.out.println(a);
        
        a -= 1; // a = a - 1
        System.out.println(a);
        
        a *= 2; // a = a * 2
        System.out.println(a);
        
        a /= 2; // a = a / 2
        System.out.println(a);
        
        a %= 2; // a = a % 2
        System.out.println(a);
    }
}

Operadores de Comparación

En Java tenemos un tipo de operadores que nos sirven para comparar variables o algunos tipos de datos (numéricos y caracteres) y devolver un valor boleano.

public class Main {
    public static void main(String[] args) {
        int a = 5;
        int b = 3;
        
        System.out.println(a == b);//Igual que
        System.out.println(a != b);//Diferente que
        System.out.println(a > b);//Mayor que
        System.out.println(a >= b);//Mayor igual que
        System.out.println(a < b);//Menor que
        System.out.println(a <= b);//Menor igual que
    }
}

Operadores Lógicos

Otro tipo de operadores son los operadores lógicos los cuales nos sirven para tomar decisiones en función de vario valores booleanos. AND (&&) el cual devuelve true solo si los dos valores son true, OR (||) el cual devulve true cuando uno de los dos valores es verdadero y NOT (!) el cual cambia al valor contrario. Además de comparar valores boleanos solo podemos también hacer operaciones lógicas complejas.

public class Main {
    public static void main(String[] args) {
        int a = 5;
        int b = 3;
        
        System.out.println(true && true);
        System.out.println(true && false);
        System.out.println(false && false);

        System.out.println(3 > 2 && 5 == 2);

        System.out.println(true || true);
        System.out.println(true || false);
        System.out.println(false || false);

        System.out.println(3 > 2 || 5 == 2);

        System.out.println(!true);
        System.out.println(!false);

        System.out.println(!(3 > 2) || 5 == 2);
    }
}

Operadores Unarios

Los últimos operadores que tenemos en Java son los Unarios. Son aquellos que utilizamos para incrementar o decrementar valores. Dependiendo de donde pongamos el operador se realizara el incremento antes o despues. 

public class Main {
    public static void main(String[] args) {
        int a = 5;
        int b = 3;
        
        System.out.println(++b);//Suma 1 y luego imprime
        System.out.println(b++);//Imprime primero y luego suma 1
        System.out.println(b);
        
        System.out.println(--b);//Resta 1 y luego imprime
        System.out.println(b--);//Imprime primero y luego resta 1
        System.out.println(b);
    }
}
