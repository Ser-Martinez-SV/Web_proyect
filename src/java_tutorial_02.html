Java: Variables y Tipos de Datos

Aprende a declarar y usar variables en Java.

Variables en Java

En Java, una variable es un contenedor que guarda un valor. Cada variable debe tener un tipo de dato que especifica el tamaño y el tipo de valores que puede almacenar.

Declaración y Asignación

Para declarar una variable, debes especificar su tipo y un nombre. Puedes asignarle un valor en el momento de la declaración o más tarde.

// Declaración de variables
int numeroEntero;
String texto;

// Asignación de valores
numeroEntero = 10;
texto = "Hola Java";

// Declaración y asignación en una sola línea
double precio = 19.99;
boolean esVerdadero = true;

Tipos de Datos Primitivos

Java tiene ocho tipos de datos primitivos:

  • byte: Números enteros pequeños (-128 a 127).
  • short: Números enteros cortos (-32,768 a 32,767).
  • int: Números enteros (el más común).
  • long: Números enteros largos.
  • float: Números de punto flotante de precisión simple.
  • double: Números de punto flotante de precisión doble (el más común para decimales).
  • boolean: Valores lógicos (true o false).
  • char: Un solo carácter.

Tipos de Datos No Primitivos (Referencia)

Los tipos de datos no primitivos se refieren a objetos y no tienen un valor predefinido. El más común es String.

String saludo = "¡Bienvenido a CodeAcademy!";
System.out.println(saludo);
