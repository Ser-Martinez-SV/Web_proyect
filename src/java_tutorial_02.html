Java: Variables y Tipos de Datos

Aprende a declarar y usar variables en Java.

Variables en Java

En Java, una variable es un contenedor que guarda un valor. Cada variable debe tener un tipo de dato que especifica el tamaño y el tipo de valores que puede almacenar.

Además de las variables aprenderemos los tipos de datos que pueden contener estas variables.
Para declarar una variable primero debemos indicar el tipo de dato, luego el nombre y el valor. Empecemos viendo los tipos de datos primitivos.

Variables numéricas

Para guardar numeros enteros contamos con 4 tipos de variable, segun la cantidad de momeoria que ocupan, es decir, que tan grande puede ser el número que guardamos ahí. [byte (-127, 127), short (-32k, 32k), int (-2B, 2B), long (-2^61, 2^61)]. Es cierto que el más comun es int.

public class Main {
    public static void main(String[] arg) {
        byte edad = 13;
        short num = 12455;
        //Podemos separar numeros grandes con barras bajas, sin que afecte en nada.
        int millones = 123_456_789; 
        //Inportantre la "L" final en estos numeros
        long trillones = 123_334_253_454L;
        //Podemos modificar el valor de una variable simplemente escribiendo su nombre y nuevo valor.
        num = 34;
    }
}

Dos maneras de guardar numeros con decimales son con los tipos de datos float y double. Siendo double capaz de guardar números más grandes que float. Y para definir float debemos poner una f al final del número, se recomienda que sea mayúscula, para diferenciarlo mejor del número en sí.

public class Main {
    public static void main(String[] arg) {
        float decimal = 1.33F; 
        double precio = 2134.5478;
    }
}

Variables Booleanas

Tambien tenemos el tipo de datos booleano, estas varaibles booleanas que pueden contener true o false (verdadero o falso)las cuales nos serán muy utiles más adelante para comprobar condiciones de estructuras de control.:

public class Main {
    public static void main(String[] arg) {
        boolean encendido = true;  
        boolean abierto = false;  
    }
}

Variables de Texto

Otro tipo son las cadenas de texto o String. Pueden estar entre comillas dobles o simples o incluso triples. Cuando utilizamos comillas triples podremos utilizar comillas dobles dentro de este.

public class Main {
    public static void main(String[] arg) {
        String texto = "Hello Poyo";    
        String frase = "Al pan pan y al vino vino"; 
        String comillas_triples = """Angel dijo: "Hugo, vete de clase". Y hugo se fue.""";    
    }
}

Variables de Caracteres

Un tipo de datos similar al de texto pero donde solo podemos guardar un único caracter (letra, numero, simbolo...). Siempre indicando su valor entre comillas simples.

public class Main {
    public static void main(String[] arg) {
        char letra = 'A'; 
        char simbolo = '-';      
    }
}

Constantes y Errores

Para terminar tenemos un concepto que aplica a todas las variables que utilizamos queremos que se mantengan con el valor inicial todo el código. A estas variables se les llaman constantes o finales. Y se declaran simplemente escribiendo final antes de su declaración normal. Su valor no podrá ser cambiado.

public class Main {
    public static void main(String[] arg) {
        final float PI = 3.14F;
        final String MI_NOMBRE = "Sergio"; 
        final char LETRA_A = 'A';
    }
}
