Java: Condiciones

Controla las diferentes estructuras de control y condiciones de Java.

Condicionales

Como en todo lenguaje de programación tenemos condicionales, partes de codigo donde se hace una comprobación para seguir un camino u otro. Esto añade lógica y complejidad a nuestro código.

If Else

Con if junto a una condición podemos ejecutar o no una parte de código. Con else lo que haremos será ejecutar indicar que parte de código se ejecutara si la condición del if no se cumple. Además no no existe elif como en python por ejemplo. Para ello se utiliza else seguido de if (else if (){} ).

public class Main {
    public static void main(String[] args) {
        System.out.println("Hola Mundo!");
    }
}

Explicación Primer Programa

Todo programa de Java tiene la misma estructura. Empieza con una clase Main y dentro de estra una función principal main también. Cuando ejecutamos un programa de Java el compilador llama a la clase Main y esta llama a su función principal (main) y entonces se ejecuta nuestro código.
