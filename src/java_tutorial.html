Aprende uno de los lenguajes de programación más populares y versátiles.

¿Qué es Java?

Java es un lenguaje de programación de propósito general, concurrente, basado en clases y orientado a objetos, que fue diseñado específicamente para tener tan pocas dependencias de implementación como fuera posible. Su lema "Write Once, Run Anywhere" (WORA) significa que el código Java compilado puede ejecutarse en todas las plataformas que soporten Java sin necesidad de ser recompilado.

Primer Programa en Java: "Hola Mundo"

El clásico programa "Hola Mundo" es el punto de partida para cualquier lenguaje de programación.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hola Mundo!");
    }
}

Variables y Tipos de Datos en Java

Java es un lenguaje fuertemente tipado, lo que significa que todas las variables deben ser declaradas con un tipo de dato específico antes de ser utilizadas.

  • int: Números enteros.
  • double: Números de punto flotante.
  • boolean: Valores verdadero/falso.
  • String: Cadenas de texto (es una clase, no un tipo primitivo).
int edad = 25;
double precio = 99.99;
boolean activo = true;
String mensaje = "Bienvenido";
public class Main {
    public static void main(String[] arg) {
        var email = "pepelopez@gmail.com";
        System.out.println(email);
        var año = 2025;
        System.out.println(año);
    }
}
