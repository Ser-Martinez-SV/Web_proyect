Java: Arrays

Los Arrays, uno de los tipos de datos más utilizados en Java.

Listas, Arrays y Matrices

Son uno de los tipos de datos más utilizados en todo lenguaje de programación, en Java tienen sus cualidades y funciones, vamos a verlos.

Definición de Arrays (Listas)

Los Arrays son secuencias de un solo tipo de dato ordenadas por indices empenzando en 0. Para definirlos tenemos dos formas, indicar su longitud o indicar todos sus valores. Debemos especificar el tipo de dato junto a un par de [ ].

class Main {
    public static void main(String[] args) {
        int[] numeros = new int[5];
        double[] decimales = new double[5];
        String[] nombres = {"Pepe", "Ana","Juan"};
        int[] numeritos = {2,3,4,5,6};
    }
}

Acceso a Datos

Como he indicado anteriormete los elementos de un array estan ordenados por índices, siendo el primero 0, por lo que para acceder a un dato, para utilizarlo o cambiar su valor, deberemos indicar el nombre de la variable junto a el índice entre [ ].

class Main {
    public static void main(String[] args) {
        int[] numeros = new int[5];
        numeros[0] = 1;
        System.out.println(numeros[0]);
    }
}

.length

.length en una método de los arrays que nos devuelve la longitud del mismo. Similar al .length() de los strings pero sin ( ).

class Main {
    public static void main(String[] args) {
        int[] numeros = new int[5];
        System.out.println(numeros.length);//5
    }
}

¿Y si queremos definir una variable en función de una condición pero darle un valor que no sea booleano?

En Java podemos hacerlo, simplemete deberemos seguir la estructura a continuación y darle un valor string, int o el que queramos. Estos trucos de Java hacen el código más limpio y corto.

public class Main {
    public static void main(String[] args) {
        int salario = 120_000;    
        String clase = salario > 100_000 ? "Clase alta" : "Clase media";
    }
}

Switch

Además de if else Java cuenta con otra estructura de control, switch (Según). Si dada una condición tenemos muchas posibilidades sería muy costos y óptimo hacer muchos else if, esta estructura de control nos permite en base a una variable realizar diferentes acciones según su valor. 

public class Main {
    public static void main(String[] args) {
        String rol = "admin";
        switch (rol){//También se puede hacer con variables númericas
            case "admin":
                System.out.println("Eres admin");
                break; //Muy importante añadir el break para que termine el switch y no siga comprobando
            case "moderador":
                System.out.println("Eres moderador");
                break;
            default://Es el valor por defecto, en el caso no coincidir con ninguno de los anteriores
                System.out.println("Eres un usuario normal.");
                break;
        }
    }
}
