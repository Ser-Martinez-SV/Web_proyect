Java: Arrays

Los Arrays, uno de los tipos de datos más utilizados en Java.

Listas, Arrays y Matrices

Son uno de los tipos de datos más utilizados en todo lenguaje de programación, en Java tienen sus cualidades y funciones, vamos a verlos.

Definición de Arrays (Listas)

Los Arrays son secuencias de un solo tipo de dato ordenadas por indices empenzando en 0. Para definirlos tenemos dos formas, indicar su longitud o indicar todos sus valores. Debemos especificar el tipo de dato junto a un par de [ ].

class Main {
    public static void main(String[] args) {
        int[] numeros = new int[5];
        double[] decimales = new double[5];
        String[] nombres = {"Pepe", "Ana","Juan"};
        int[] numeritos = {2,3,4,5,6};
    }
}

Acceso a Datos

Como he indicado anteriormete los elementos de un array estan ordenados por índices, siendo el primero 0, por lo que para acceder a un dato, para utilizarlo o cambiar su valor, deberemos indicar el nombre de la variable junto a el índice entre [ ].

class Main {
    public static void main(String[] args) {
        int[] numeros = new int[5];
        numeros[0] = 1;
        System.out.println(numeros[0]);
    }
}

.length

.length en una método de los arrays que nos devuelve la longitud del mismo. Similar al .length() de los strings pero sin ( ).

class Main {
    public static void main(String[] args) {
        int[] numeros = new int[5];
        System.out.println(numeros.length);//5
    }
}

Arrays.toString(array)

En Java existe una librería muy comun que se llama Arrays y nos permite realizar diferentes funciones con arrays. Simplemente debemos importarla como vemos en el ejemplo y una función muy útil es Arrays.toString(array) la cual nos permite imprimir arrays por consola.

import java.util.Arrays;
class Main {
    public static void main(String[] args) {
        int[] numeros = {2,3,4,5,6};
        System.out.println(Arrays.toString(numeros));
        //Salida: [2,3,4,5,6]
    }
}

Arrays.sort(array)

Dentro de esta misma librería tenemos otra función muy útil, la cual nos permite ordenar arrays de manera ascendente.

import java.util.Arrays;
class Main {
    public static void main(String[] args) {
        int[] numeritos = {9,7,4,2,6};
        Arrays.sort(numeritos);
        System.out.println(Arrays.toString(numeritos));
        //Salida: [2,4,6,7,9]
    }
}

Arrays.equals(array1, array2)

Al igual que pasa con las strings, no podemos comparar dos arrays con el operador == si no que tenemos que utilizar una función de esta librería, la cual devuelve true solo si todas las posiciones de los arrays coinciden.

import java.util.Arrays;
class Main {
    public static void main(String[] args) {
        int[] numeros = new int[5];
        int[] numeritos = {9,7,4,2,6};
        System.out.println(Arrays.equals(numeros, numeritos));
        //Salida: false
    }
}

Arrays.fill(array, value)

Por último tenemos una función que nos permite rellenar un array con un mismo valor en todos sus índices.

import java.util.Arrays;
class Main {
    public static void main(String[] args) {
        int[] numeros = new int[5];
        Arrays.fill(numeros, 24);
        System.out.println(Arrays.toString(numeros));
        //Salida: [24,24,24,24,24]
    }
}

Matrices

Un tipo más complejo de arrays son las matrices, que no son más que arrays de 2 dimensiones, que utilizan 2 índices. Lo podemos entender también como si fuera una lista de listas y que el primer índice indica a que lista nos referimos y el segundo a que valor de la lista.

Para declararlas utilizamos las mismas estructuras que los arrays pero con dos []. Y para acceder a los valores igual, indicamos los 2 índices del valor entre dos diferentes [ ].

import java.util.Arrays;
class Main {
    public static void main(String[] args) {
        int [][] matriz2x3 = new int[2][3]; 
        int[][] matriz = {
            {1,2,3},
            {3,5,6},
            {7,8,9}
        };//Podemos declararla en una sola linea, pero hecho así se ve más claro
        matriz[2][2] = 0;
    }
}

Arrays.deepToString(matriz)

Al igual que con los arrays normales hay algunas funciones de la librería de Arrays que podemos utilizar, en este caso podemos imprimirlas por consola.

import java.util.Arrays;
class Main {
    public static void main(String[] args) {
        int[][] matriz = {
            {1,2,3},
            {3,5,6},
            {7,8,9}
        };
        System.out.println(Arrays.deepToString(matriz));
        //Salida: [[1, 2, 3], [3, 5, 6], [7, 8, 9]]
    }
}
