Tutorial de CSS

Estiliza tus páginas web con hojas de estilo en cascada.

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento escrito en HTML. CSS describe cómo los elementos deben ser mostrados en pantalla, en papel, o en otros medios.

Sintaxis Básica de CSS

Una regla CSS consiste en un selector y un bloque de declaración. El selector apunta al elemento HTML que deseas estilizar, y el bloque de declaración contiene una o más declaraciones separadas por punto y coma. Cada declaración incluye una propiedad CSS y un valor, separados por dos puntos.

p {
    color: blue;
    font-size: 16px;
}

Selectores Comunes de CSS

Los selectores se utilizan para "encontrar" (seleccionar) los elementos HTML a los que deseas aplicar un estilo.

  • Selector de elemento: Selecciona elementos por su nombre (ej. p, h1).
  • Selector de ID: Selecciona un elemento con un ID específico (ej. #miId).
  • Selector de clase: Selecciona elementos con una clase específica (ej. .miClase).
  • Selector universal: Selecciona todos los elementos (ej. *).
