Tutorial de JavaScript

Añade interactividad y dinamismo a tus páginas web.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite implementar funciones complejas en páginas web. Cada vez que una página web hace algo más que simplemente mostrar información estática (mostrar actualizaciones de contenido a tiempo, mapas interactivos, gráficos animados 2D/3D, etc.), puedes apostar que JavaScript está involucrado.

Variables y Tipos de Datos

En JavaScript, puedes declarar variables usando let, const o var. Los tipos de datos incluyen números, cadenas de texto, booleanos, objetos, etc.

let nombre = "Juan"; // Cadena de texto
const edad = 30; // Número
let esEstudiante = true; // Booleano

console.log(nombre, edad, esEstudiante);

Funciones en JavaScript

Las funciones son bloques de código diseñados para realizar una tarea particular. Una función se ejecuta cuando "algo" la invoca (la llama).

function saludar(nombre) {
    console.log("Hola, " + nombre + "!");
}

saludar("María"); // Llama a la función
