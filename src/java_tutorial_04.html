Java: Strings

Las strings son uno de los tipos de datos más usados en Java, por lo que es indispensable saber utilizarlas correctamente.

Strings

Las cadenas de texto en Java tiene comportamientos diferentes al resto de tipos de datos, además de muchas funciones que aprenderemos a utilizar en esta lección.

Concatenación

Un comportamiento muy comun que vamos a utilizar a menudo es la concatenación de cadenas de text, es muy sencilla, utilizamos el operador +. Podemos concatenar con otros tipos de datos como numeros con el mismo operador. No obstante tenemos una función para concatenar strings.

.length()

Esta función nos devuelve la longiud de la cadena de texto indicada enteriormente.

.charAt(n) .indexOf(str)

charAt(n), devuelve el caracter de la posición n. Debemos tener en cuenta que la primera posición es la 0 y que si no encuentra el caracter en el string devolvera -1.

indexOf(str), nos deveulve la posición de dicho str. Si buscas un str que no esta devolvera -1. También podemos indicar desde que cadena queremos que empiece a buscar.

lastIndexOf(str), es una variante de indexOf pero que devuelve la posición de la ultima aparicion de ese str.

.substring(n, m)

Devuelve una subcadena desde la primera posición indicada a la ultima, de n a m.

.equals(str)

Sirve para compara dos str, en Java no podemos comparar strings con ==, debemos hacerlo así. Devuelve un valor booleano.

Existe una variante que es .equalsIgnoreCase(str) la cual funciona igual solo que no tiene en cuenta mayúsculas ni minúsculas.

.replaceAll(str1, str2)

Remplaza todos los primeros str indicados por los segundos. No modifica la str original, devuelve un str nuevo.

.replaceAll(str1, str2)

Remplaza todos los primeros str indicados por los segundos. No modifica la str original, devuelve un str nuevo.
