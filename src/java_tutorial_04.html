Java: Strings

Las strings son uno de los tipos de datos más usados en Java, por lo que es indispensable saber utilizarlas correctamente.

Strings

Las cadenas de texto en Java tiene comportamientos diferentes al resto de tipos de datos, además de muchas funciones que aprenderemos a utilizar en esta lección.

Concatenación

Un comportamiento muy comun que vamos a utilizar a menudo es la concatenación de cadenas de text, es muy sencilla, utilizamos el operador +. Podemos concatenar con otros tipos de datos como numeros con el mismo operador. No obstante tenemos una función para concatenar strings.

public class Main {
    public static void main(String[] args) {
        String texto =  "Hello wordl";
        texto = texto + "!!";
        texto = texto + " " + 34;
        texto.concat(". Poyo");
        System.out.println(texto);//Hello wordl!! 34. Poyo
    }
}

.length()

Esta función nos devuelve la longiud de la cadena de texto indicada enteriormente.

public class Main {
    public static void main(String[] args) {
        String texto =  "Hello wordl!!";
        System.out.println(texto.length());
    }
}

.charAt(n) .indexOf(str)

charAt(n), devuelve el caracter de la posición n. Debemos tener en cuenta que la primera posición es la 0 y que si no encuentra el caracter en el string devolvera -1.

indexOf(str), nos deveulve la posición de dicho str. Si buscas un str que no esta devolvera -1. También podemos indicar desde que cadena queremos que empiece a buscar.

lastIndexOf(str), es una variante de indexOf pero que devuelve la posición de la ultima aparicion de ese str.

public class Main {
    public static void main(String[] args) {
        String texto =  "Hello wordl!!";
        System.out.println(texto.charAt(0));
        System.out.println(texto.indexOf("H"));
        System.out.println(texto.lastIndexOf("l"));
    }
}

.substring(n, m)

Devuelve una subcadena desde la primera posición indicada a la ultima, de n a m.

public class Main {
    public static void main(String[] args) {
        String texto =  "Hello wordl!!";
        System.out.println(texto.substring(5, 7));
    }
}

.equals(str)

Sirve para compara dos str, en Java no podemos comparar strings con ==, debemos hacerlo así. Devuelve un valor booleano.

Existe una variante que es .equalsIgnoreCase(str) la cual funciona igual solo que no tiene en cuenta mayúsculas ni minúsculas.

public class Main {
    public static void main(String[] args) {
        String texto =  "Hello wordl";
        System.out.println(texto.equals("Hello wordl 1"));
        System.out.println(texto.equalsIgnoreCase("hello Wordl"));
    }
}

.replaceAll(str1, str2)

Remplaza todos los primeros str indicados por los segundos. No modifica la str original, devuelve un str nuevo.

public class Main {
    public static void main(String[] args) {
        String texto =  "Hello wordl";
        String texto2 = texto.replaceAll("!", "*");
        System.out.println(texto2);
    }
}

.toLowerCase() .toUpperCase()

toLowerCase(), devuelve la cadena con todo minúsculas, una cadena nueva, no modifica el primer str.

toUpperCase(), devuelve la cadena con todo mayúsculas, una cadena nueva, no modifica el primer str

public class Main {
    public static void main(String[] args) {
        String texto =  "Hello wordl";
        System.out.println(texto.toLowerCase());
        System.out.println(texto.toUpperCase());
    }
}

.trim()

Elimina los espacios al principio y al final del str, ej: " hola " -> "hola". Muy util cuando leamos información del usuario por consola, ya que evita que se equivoque y escriba esacios al final sin querer.

public class Main {
    public static void main(String[] args) {
        String texto =  "Hello wordl    ";
        System.out.println(texto.trim());
    }
}

.split(delimitador)

Deveulve un array (lista) con los elemtos del string separados por el elememnto indicado.

public class Main {
    public static void main(String[] args) {
        String texto =  "Hello wordl";
        String palabras[] = texto.split(" ");
    }
}

.contains(str) .startsWith(str) .endsWith(str)

contains(str) devuelven true o false segun si contiene el texto indicado.

startsWith(str), devuelven true o false segun si empieza por el texto indicado.

endsWith(str), devuelven true o false segun si termina por el texto indicado.

public class Main {
    public static void main(String[] args) {
        String texto =  "Hello wordl!";
        System.out.println(texto.contains("Hello"));
        System.out.println(texto.startsWith("H"));
        System.out.println(texto.endsWith("!"));
    }
}

.isEmpty()

Devuelven true o false segun si la cadena esta vacia.

public class Main {
    public static void main(String[] args) {
        String texto =  "";
        System.out.println(texto.isEmpty());
    }
}

.toCharArray()

Convierte un string en un array (lista) de caracteres.

public class Main {
    public static void main(String[] args) {
        String texto =  "Hello wordl";
        char[] letras = texto.toCharArray();
        System.out.println(letras);
    }
}

String.join(delimitador, lista)

Une un array de Strings en uno solo juntando los elementos con el limitador entre estos.

public class Main {
    public static void main(String[] args) {
        String [] palabras = {"Hola", "planeta", "Tierra"};
        System.out.println(String.join(" ", palabras));
    }
}

System.out.printf()

Existe una manera de "formatear" cadenas de texto cuando las imprimimos por pantalla, es decir, indicar en un lugar de una cadena diferentes connotaciones de manera que posteriormente se indique un valor ahí.

Estas connotaciones van siempre detrás de %, entre otras son importantes:

  • %n: Para un salto de línea.
  • %b: Para valores Booleanos.
  • %s: Para string
  • %d: Para double
  • %f: Para floats.
  • %g: Para números int, byte,short, long.
  • %t: Para tiempo y fecha.
public class Main {
    public static void main(String[] args) {
        
        int numero42 = 42;
        String texto2 = "Hello, World!";
        double decimal = 3.14159;
        
        System.out.printf("The number is: %d%n", numero42);
        
        System.out.printf("The text is: %s%n", texto2);
        
        System.out.printf("The value of pi is approximately: %.2f%n", decimal);
    }
}

Otras funcionalidades

Para terminar vamos a ver unos trucos de las strings, unas secuencias que nos ayudan a indicar diferentes secuencias que no son lo que estamos poniendo, como un salto de línea, una tabulación y demás.

public class Main {
    public static void main(String[] args) {
        String comillas = "lo hizo \"pa\" nada ";//Permite comillas dentro de una str
        String barra = "Sergio\\M"; //Realmente solo aparecera una barra
        String salto = "\n";//Salto de linea
        String tabulacion = "\t";//Tabulación
    }
}
